LISTA DE EXERCICIO - TECNOLOGIA PARA FRONT-END

1. O HTML é uma linguagem de programação, onde é usada para desenvolver páginas da web, onde ele procura estruturar o conteúdo da pagina Web, sendo capaz de organizar texto, imagens, vídeos, links formulários e outras ferramentas. 

2. A linguagem de programação HTML foi desenvolvida em 1989 por Tim Berners-Lee e seus colaboradores no CERN (Conselho Europeu de Pesquisas Nucleares).

3. O HTML estrutura o conteúdo do documento sendo programado, já o CSS é uma linguagem que permite que o programador aplique estilos seletivamente a elementos em documentos HTML

4. A tag HTML serve para delimitar toda a pagina dentro de um código feito com a devida linguagem.

5. A tag <head> serve para configurar outras coisas que não seja o conteúdo da página, como por exemplo o título, já a tag <body> serve para inserir os elementos visuais nas páginas.

6. A tag  <title> serve para inserir um título na janela ou na aba da página.

7. <html>
 	<head>
 	<title> This is your Title</title>
 	</head>

 	<body>
 		<p> random stuff.</p>
 		<p> 	random stuff pt 2.</p>
 	</body>
</html>
